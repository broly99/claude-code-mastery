# 개발자 웹이력서 프로젝트 로드맵

## 프로젝트 개요

개인 개발자 포트폴리오 및 이력서를 웹으로 표현하는 반응형 웹사이트

## 기술 스택

| 분류 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3, TailwindCSS |
| 인터랙션 | JavaScript (Vanilla) |

---

## 이력서 섹션 구성

### 1. 헤더 (Header)
- 이름, 직함
- 프로필 사진
- 연락처 정보 (이메일, 전화번호)
- 소셜 링크 (GitHub, LinkedIn)

### 2. 자기소개 (About)
- 간단한 소개글
- 핵심 역량 키워드

### 3. 기술 스택 (Skills)
- 프로그래밍 언어
- 프레임워크 / 라이브러리
- 도구 및 기타

### 4. 경력 사항 (Experience)
- 회사명, 직책, 근무기간
- 주요 업무 및 성과

### 5. 프로젝트 (Projects)
- 프로젝트명
- 사용 기술
- 프로젝트 설명
- 링크 (GitHub, 데모)

### 6. 학력 (Education)
- 학교명, 전공, 졸업년도

### 7. 푸터 (Footer)
- 저작권 정보
- 추가 연락처
